基于合宙air202/208/800/801/720/H/D/G/U 系列模块的开源DTU/RTU解决方案。
支持TCP/UDP socket,支持HTTP,MQTT,等常见透传和非透传模式
支持OneNET,阿里云，百度云，腾讯云等常见公有云。
支持RTU主控模式
支持数据流模版
支持消息推送(电话，短信，网络通知)
支持GPS数据以及相关数据采集
支持ADC,I2C等外设，可以方便的扩展为屏幕、二维码等解决方案.
